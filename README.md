# ARV Property Analyzer

A property valuation tool that calculates After Repair Value (ARV) 
and Maximum Allowable Offer (MAO) to help real estate investors 
quickly underwrite deals.

## What it does
- Analyzes on-market and off-market properties
- Calculates ARV based on comparable sales and market trends
- Generates Maximum Allowable Offer figures for investment decisions
- Pulls comparable properties to support valuation

## Why I built it
Built as an independent version of a similar tool in my portfolio, 
this was developed through Perplexity to explore a different 
development approach to the same problem — giving real estate 
investors a faster way to evaluate deals without relying on 
expensive third-party platforms.
